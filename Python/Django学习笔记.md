# 项目创建
```
django-admin.py startproject myblog
```
### 项目目录结构
```
my_blog
├── manage.py   //Boss级别的管理工具，用来添加模块app等系统功能
└── my_blog     //核心代码，控制该问层面的设置
    ├── __init__.py     
    ├── settings.py     //
    ├── urls.py         //
    └── wsgi.py         //

```
